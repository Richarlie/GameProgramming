# 希尔排序
**主要思路：**这个排序实际就是直接插入排序的升级版，只不过是把其中的"1"变成"d"罢了
```C++
void Sort(vector<int>&arr){
	int d=arr.size()/2;
	while(d>=1){
		for(int i=1;i<arr.size();i++){
			int temp=arr[i];
			int j=i-d;
				while(j>=0&&arr[j]>temp){
					arr[j+d]=arr[j];
					j-=d;
				}
			arr[j+d]=temp;
		}
		for(int i=0;i<arr.size();i++){
			cout<<arr[i]<<" ";
		}
		cout<<endl;
		
		d/=2;
	}
}
```
