# 希尔排序
**主要思路：**这个排序实际就是直接插入排序的升级版，只不过是把其中的"1"变成"d"罢了
```C++
void shellSort(int arr[], int n) {
    int d = n / 2; // 初始增量
    while (d >= 1) {
        for (int i = d; i < n; i++) {
            int temp = arr[i];
            int j = i - d;
            while (j >= 0 && arr[j] > temp) {
                arr[j + d] = arr[j];
                j -= d;
            }
            arr[j + d] = temp;
        }
        // 输出每趟排序的结果
        for (int i = 0; i < n; i++) {
            cout << arr[i];
            if (i < n - 1) {
                cout << " ";
            }
        }
        cout << endl;
        d = d / 2; // 减小增量
    }
}
```
