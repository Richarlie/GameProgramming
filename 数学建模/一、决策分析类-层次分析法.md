# 一、决策分析类
## 1.层次分析法
**需求**：根据“流量 质量 颜值”等指标去评价一个明星的价值
可以对指标按照其重要性赋予权重，但问题是权重的赋予通常过于主观

![输入图片说明](/imgs/2025-04-29/8Nr0HwmWY5k1cWLR.png)
**如何客观设置权重？**此时层次分析法就起作用了
`归化处理`+`权重处理`
`归化处理`：把每个指标变成0~1之间的数值


**基本步骤**：
1. 建立
2. 模型
3. 构造各层次中的所有判断矩阵
4. 一致性检验
5. 求权重后评价
