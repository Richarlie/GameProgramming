# 决策分析类
## 1.层次分析法
譬如根据“流量 质量 颜值”等指标去评价一个明星的价值
`归化处理`+`权重处理`
`归化处理`：把每个指标变成0~1之间的数值
如何处理好权重？此时层次分析法就起作用了
**基本步骤**：
1. 建立递阶层次结构模型
2. 构造各层次中的所有判断矩阵
3. 一致性检验
4. 求权重后评价
