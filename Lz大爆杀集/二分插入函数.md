# 二分插入函数&二分查找
- 无STL
```C++
#include <vector>

template<typename T>
void binary_insert_manual(std::vector<T>& vec, const T& item) {
    if (vec.empty()) {
        vec.push_back(item);
        return;
    }//如果是空，那就直接加进去
    
    size_t left = 0;
    size_t right = vec.size();
    
    // 二分查找确定插入位置
    while (left < right) {
        size_t mid = left + (right - left) / 2;
        //优化了“(left+right)/2”可能使程序陷入死循环的弊端
        if (vec[mid] < item) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    // 在找到的位置插入元素
    vec.insert(vec.begin() + left, item);
}
```
- 有STL
``` C++
#include <vector>
#include <algorithm> // 用于std::lower_bound

// 使用二分查找确定插入位置并插入元素
template<typename T>
void binary_insert(std::vector<T>& vec, const T& item) {
    // 使用lower_bound找到第一个不小于item的位置
    auto it = std::lower_bound(vec.begin(), vec.end(), item);
    // 在该位置插入元素
    vec.insert(it, item);
}
```
- 基本的二分查找函数

`low_bound`：标准库中的二分查找算法，返回第一个**不小于**item的元素位置

`binary_search`：C++标准库`<algorithm>`头文件中提供的一个二分查找算法，用于在已排序的范围内检查特定元素是否存在。
