# 数据的输入 (25.4.14)
### 1.getline函数(string)
`getline`是C++中用于从输入流读取一行文本的常用函数。它有两种主要形式
```C++
-1-
#include <iostream>
#include <string>
std::string line;
std::getline(std::cin, line);  // 从标准输入读取一行到字符串line
//左来源，右去处 
-2-
#include <iostream>
char buffer[256];
std::cin.getline(buffer, 256);  // 从标准输入读取最多256个字符到buffer
//直接表明读取目的地和数量
```

### 3.有输入，直到遇到空行则停止输入
```C++
string str;
	while(getline(cin,str)){
		if(str.empty()){
			break;
		}
	istringstream iss(str);
	int a,b;
	if(iss>>a>>b){
		cout<<a<<" "<<b<<endl;
	}
	}
```
