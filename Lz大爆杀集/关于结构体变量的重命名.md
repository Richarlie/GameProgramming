# 关于结构体变量的重命名
- 无指针：
```C++
struct person{
	string name;
	int number;
	};
typedef struct person personalias;
```
```C++
 typedef struct person{
	string name;
	int number;
	}personalias;
```

- 有指针：
```C++
struct person{
	string name;
	int number;
	
	};
typedef struct person personalias;
```
