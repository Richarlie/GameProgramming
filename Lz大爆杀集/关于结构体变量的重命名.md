# 关于结构体变量的重命名
- 无指针：
```C++
struct person{
	string name;
	int number;
	};
typedef struct person personalias;
```
```C++
 typedef struct person{
	string name;
	int number;
	}personalias;
```

- 有指针：
```C++
struct person{
	string name;
	int number;
	person* next;
	};
typedef struct person personalias;
typedef personalias* personptr;
```
```C++
typedef struct person{
	string name;
	int number;
	person* next;
	}personalias,*personptr;
```
- 在结构体中的为什么使用   Person*  而不是   PersonAlias   或   PersonPtr  ？因为未被定义
在结构体定义内部，你必须使用结构体的原始名称（或其未修饰的类型名）来声明成员，而不是使用通过   typedef   或   using   创建的别名。
