# 分数拆分
注意变量数值别溢出了！
### 描述：

输入一个正整数，找到所有的正整数x>=y,使得1/k=1/x+1/y

### 输入格式：

一个正整数n，n<=100000

### 输出格式：

一行一个分式等式  
小的分数显示在前面；多个等式，等式右边第一个分数小的先输出

### 输入样例：

2

### 输出样例：

1/2=1/6+1/3  
1/2=1/4+1/4

```C++
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int k;
    cin >> k;
    long long k_squared = (long long)k * k;
    vector<pair<long long, long long>> solutions;

    for (int d = 1; d <= k; ++d) {
        if (k_squared%d==0) {
            long long d2 = d;
            long long d1 = k_squared / d2;
            if (d1 >= d2) {
                long long x = d1 + k;
                long long y = d2 + k;
                solutions.emplace_back(y, x);
            }
        }
    }

    sort(solutions.begin(), solutions.end());

    for (const auto& sol : solutions) {
        long long y = sol.first;
        long long x = sol.second;
        cout << "1/" << k << "=1/" << x << "+1/" << y << endl;
    }

    return 0;
}
```
