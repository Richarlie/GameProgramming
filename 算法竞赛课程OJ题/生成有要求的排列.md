# 生成有要求排列
### 描述：
从键盘输入N个整数N<=10,要求输出这N个整数的全排列，且每一个排列当中，相邻两个数不能同为奇数。

### 输入格式：
第一行一个正整数N  
第二行N个整数，以空格分隔

### 输出格式：
一行一个排列，按小到大的顺序输出

### 输入样例：
3  
1 1 2
### 输出样例：
1 2 1

**基本思路：**在生成排列的基础上加了一个验证机制（ha）

```C++
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

// 检查相邻数字是否都是奇数
bool isValid(const vector<int>& nums) {
    for (size_t i = 0; i < nums.size() - 1; ++i) {
        if (nums[i] % 2 == 1 && nums[i + 1] % 2 == 1) {
            return false; // 如果相邻数字都是奇数，返回false
        }
    }
    return true; // 否则返回true
}

int main() {
    int N;
    cin >> N; // 输入整数个数N

    vector<int> nums(N);
    for (int i = 0; i < N; ++i) {
        cin >> nums[i]; // 输入N个整数
    }

    // 对输入的整数进行排序，确保生成的排列是按字典序排列的
    sort(nums.begin(), nums.end());

    // 使用do-while循环生成并输出所有满足条件的排列
    do {
        if (isValid(nums)) { // 检查当前排列是否满足条件
            for (int i = 0; i < N; ++i) {
                cout << nums[i] << (i == N - 1 ? "\n" : " "); // 输出排列，最后一个数字后换行
            }
        }
    } while (next_permutation(nums.begin(), nums.end())); // 生成下一个排列

    return 0;
}
```
