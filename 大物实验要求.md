# 大物实验要求
## 一、在实验室中的要求
1. 不做与实验无关的事情
2. 在规定时间内完成实验
3. 实验数据符合预期或在允许的误差范围内
## 二、评分标准
总评成绩=考勤分+实验操作分+实验报告分
## 三、关于预习部分
1. 实验名称
2. 实验目的
3. 实验仪器
4. 实验原理（概括归纳应包括：“基本原理”“基本公式”“基本示意图”）
5. 实验步骤
>当然，每个实验可按照课程伴侣上的要求做好预习
## 四、关于实验部分
1. 记录数据的表格
2. 结果处理和作图
3. 思考题
4. 实验总结
